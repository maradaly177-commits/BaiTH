# Website Quản Lý Danh Mục Và Sản Phẩm

##  Giới thiệu
Đây là website quản lý danh mục và sản phẩm được xây dựng bằng Laravel.  
Hệ thống cho phép thêm, sửa, xóa và quản lý danh mục, sản phẩm.

##  Công nghệ sử dụng
- Laravel
- PHP
- MySQL
- Bootstrap
- Blade Template

##  Chức năng chính
- Quản lý danh mục (CRUD)
- Quản lý sản phẩm (CRUD)
- Upload hình ảnh sản phẩm
- Phân trang danh sách
- Validation dữ liệu

##  Cài đặt
1. Clone project
2. Chạy `composer install`
3. Copy `.env.example` thành `.env`
4. Cấu hình database
5. Chạy `php artisan migrate`
6. Chạy `php artisan serve`
